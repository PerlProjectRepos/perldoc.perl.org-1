NAME

App::Prove::State - State storage for the http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd"> http://www.w3.org/1999/xhtml"> perltidy

perltidy

prove command.

VERSION

Version 3.36

DESCRIPTION

The http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd"> http://www.w3.org/1999/xhtml"> perltidy

perltidy

prove command supports a http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd"> http://www.w3.org/1999/xhtml"> perltidy

perltidy

--state option that instructs it to store persistent state across runs. This module implements that state and the operations that may be performed on it.

SYNOPSIS

  1.     # Re-run failed tests
  2.     $ prove --state=failed,save -rbv
